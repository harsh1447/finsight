# FinSight

AI-Powered Personal Finance Manager

## Problem

People struggle to track where their money goes every month because transactions are spread across UPI apps, bank accounts, cards, and subscriptions.

## Features

- Expense Tracking
- Income Tracking
- Budget Management
- Analytics Dashboard
- AI Insights
- Financial Chatbot

## Tech Stack

### Frontend
- React
- Tailwind CSS

### Backend
- Spring Boot
- Spring Security
- JWT

### Database
- PostgreSQL

### AI
- Gemini API
